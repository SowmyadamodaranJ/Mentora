# AI-Interview-Preparation

Local build and development instructions for this project.

## Prerequisites

- Ensure you have Node.js (v18+) and npm installed on your machine.

## Setup Instructions

1. **Install Dependencies**
   Run the following command to download all required packages:
   ```bash
   npm install
   ```

2. **Start Development Server**
   Start the local development server to view the application in your browser:
   ```bash
   npm run dev
   ```

3. **Build for Production**
   If you want to build the project into a production-ready static site:
   ```bash
   npm run build
   ```

## supabase

This project uses Supabase. Configure your `.env` variables if necessary to point to your Supabase instance to utilize the backend functionality.
